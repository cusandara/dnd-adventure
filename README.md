# DnD 5e Adventure Sim

A text-based, Choose-Your-Own-Adventure style RPG built with Next.js and React.

## Features
- **Character Creation**: Roll for stats, select Class and Race (PHB compliant).
- **Combat System**: Turn-based combat with Initiative, AC, Attack Rolls, and Damage.
- **Quest System**: Dynamic quest generation and tracking.
- **Inventory & Shop**: Buy/Sell items, manage gold, and equip gear.
- **Non-Combat Encounters**: Skill checks, roleplay scenarios, and dynamic consequences.

## Getting Started

1.  Refer to `DEPLOYMENT.md` for deployment instructions.
2.  Run locally:
    ```bash
    npm install
    npm run dev
    ```
